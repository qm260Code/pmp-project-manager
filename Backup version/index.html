    项目情况总览 (Dashboard)

    快速洞察项目进展、绩效指标及核心高风险要素

    📅 里程碑进度条 最近更新

    ⚠️ 核心风险警示

    PMP 5大过程组 10大知识领域矩阵

    点击过程领域模块，录入与更新对应的项目文档交付物

    相关方登记册 (Stakeholder Register)

    规划并更新利益相关方的权力与利益，实施针对性参与策略

    📊 权力/利益矩阵 (Power/Interest Grid)

    自动归类相关方，指导沟通干预策略

    📋 登记明细表

    姓名 角色/职责 权力/利益 参与状态 操作

    风险登记册 (Risk Register)

    识别并跟踪项目潜在威胁，配置应对措施与责任人

    🔥 5x5 风险热力图 (Probability/Impact Matrix)

    点击网格可筛选出对应的风险条目

    📋 风险评估明细表

    风险描述 类型 P x I = 得分 等级 应对策略 / 责任人 操作

    进度里程碑与工作网络甘特图

    管理活动起止时间与前置依赖，图形化展示项目网络生命周期

    📈 动态 SVG 甘特图 (Gantt Chart)

    📋 活动与任务分解表

    活动/里程碑名称 时间跨度 进度 (%) 负责人 依赖前置 操作

    成本估算与挣值管理系统 (EVM)

    对标计划价值(PV)、实际成本(AC)与挣值(EV)，预测项目完工情况

    📋 项目成本工作包明细表

    费用项/工作包描述 计划价值 (PV) 实际成本 (AC) 挣值比例 (EV) 操作

    责任分配矩阵 (RACI Matrix)

    定义并维护项目各角色在核心活动中的职责：负责(R)、执行/批准(A)、咨询(C)、知会(I)

    团队组织架构 (Project Org Chart)

    记录项目成员汇报关系，并图形化展现树状层级拓扑图

    📊 组织汇报架构树 (SVG Hierarchy Tree)

    📋 项目成员登记明细

    姓名 角色与部门 汇报给 操作

    项目待完成事项 (Action Items Tracker)

    跟踪并记录待办工作包、起止日期、优先级、逾期及责任接口人

    待完成事项内容 触发日期 负责人 期望完成日期 延迟天数 优先级 当前状态 操作

    数据交互与外部分析报告

    在此将项目进度与绩效指标导出，以便交付相关方或分析决策

    📥 数据备份与迁移

    推荐您定期导出项目数据 JSON 格式进行本地物理备份。本应用不包含云后台，清除浏览器缓存将导致未导出的项目数据丢失。

    📊 CSV 报表批量导出

    用于在 Excel 中做深度数据分析或制作图表

    📄 汇报专用：项目状况分析大报告

    系统将聚合整个项目的管理计划、里程碑甘特图、EVM挣值图以及高优先级风险，生成一份符合 PMP 标准的精美纸面汇报文档。

    💡 提示：在点击“生成并打印”后，在系统的打印对话框中选择“保存为 PDF”，即可导出为项目周报/月报交付件。
